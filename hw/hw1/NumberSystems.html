Machine language is the lowest level language that the computer uses. It is writtten in binary code, which is a numbering system that uses only ones and zeroes. It can be described as 2N where N = the digit, and is referred to as a base2 system.

Is binary the most efficient number system? No. So why do we use it for computers? We use it because compters run on electricity, and switches can be on or off, so a number system with only two digits works best for this.
